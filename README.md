# homebrew-versions
A cask-versions for the Homebrew that provides by michilu.

# Install

```console
$ brew tap michilu/homebrew-versions git@github.com:michilu/homebrew-versions.git
```

# Development

```console
$ cd $(brew --repository)/Library/Taps/michilu/homebrew-versions
```
